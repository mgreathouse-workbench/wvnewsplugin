## Obituaries — now fully wired for print

- **Backend**: marketplace obits feed joins to the live obit doc, filters by `printRunDate` (so you can pull "obits for the July 10 edition"), and sends `photoUrl`. Back-compat: obits without a run date still show for all editions.
- **Plugin**: obits build as structured blocks in one threaded column; each portrait is **anchored inline at the top of its obit** (column-wide, portrait ratio, fill-crop). Non-fatal — an image failure still places the text.
- **To test**: load `dist/wvnews-print-0.1.1.ccx` in InDesign → pull obits for an edition.
